Detailed Description

The ConfigManager class.

This class allows for reading from and writing to an XML configuration file. Configuration files are useful for storing values that may change frequently, such as controller coefficients and physical device dimensions. Variables are represented as a two-level tree of categorized variables. This allows for duplicating variable names without ambiguity.
